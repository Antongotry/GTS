# Инструкция по деплою

## ⚠️ ВАЖНО: Порядок работы

**ВСЕГДА сначала пушим в GitHub, потом автодеплой сам обновит файлы на сервере!**

### Правильный порядок:

1. **Делаешь изменения** в файлах
2. **Компилируешь SCSS:** `npm run compile:css` или `npx sass sass/style.scss:style.css`
3. **Коммитишь и пушишь:**
   ```bash
   git add .
   git commit -m "Описание изменений"
   git push origin main
   ```
4. **Ждёшь 1-2 минуты** - автодеплой обновит файлы на сервере автоматически

### ❌ НЕ ДЕЛАЙ ТАК:

- НЕ заливай файлы через SSH напрямую (они перезапишутся при следующем автодеплое)
- НЕ редактируй файлы на сервере напрямую

### Если нужно срочно залить файлы:

Только в экстренных случаях можно залить через SSH, но потом ОБЯЗАТЕЛЬНО:
1. Закоммить эти же изменения локально
2. Запушить в GitHub
3. Иначе они пропадут при следующем автодеплое!
